Покладіть файл фонового зображення сюди і перейменуйте його:

  → bg.jpg    (рекомендовано, JPEG/WebP для меншого розміру)
  → bg.png    (якщо потрібна прозорість)
  → bg.webp   (найкращий варіант для веб)

Рекомендовані параметри зображення:
  - Розмір: 1920×1080 або більше (Full HD / 4K)
  - Темне зображення (overlay накладеться поверх)
  - Приклади підходящих фонів:
      • Скриншот CS2 карти зверху
      • Абстрактний темний фон
      • Tactical / military текстура

Підтримувані формати: jpg, jpeg, png, webp, avif

Після завантаження:
  Відкрийте styles/base.css і розкоментуйте рядок:
    background-image: url('../images/bg.jpg');
  Замінивши bg.jpg на ім'я вашого файлу.
